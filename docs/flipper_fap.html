Browser FAP uploader

Connect the Flipper in CDC mode, fetch the newest release asset for your firmware track, and upload it through WebSerial without qFlipper.

Flash app

Ready. Fetch a release or pick a local file.

How it works

Flipper must expose the CLI prompt over USB. Set Settings -> Power -> USB mode -> CDC first.

  • Use Chrome or Edge on desktop. WebSerial needs HTTPS or localhost.
  • Click Connect and pick the Flipper CDC port.
  • Click Fetch latest release or choose a local .fap.
  • Click Upload. The page writes the file with storage write_chunk.
  • The default install path is /ext/apps/GPIO/c5lab_dev.fap.
